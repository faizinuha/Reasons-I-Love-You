# 💕 Reasons I Love You

A romantic interactive Valentine's website with AI-powered features to express your feelings in a creative and fun way.

## 🎁 What's This?

This is a special surprise website designed to confess feelings and ask someone to be your Valentine. It features:

- 💌 Interactive "Will you be my Valentine?" game
- ✨ AI-powered love letter generator (Gemini AI)
- 🎯 Creative date idea suggestions
- 📸 Beautiful photo gallery
- ⏱️ Relationship timer
- 🎵 Background music
- 🎉 Confetti animations

## 🚀 How to Use

### Option 1: Simple Version (index.html)
Just open `index.html` in a browser - no setup needed!

### Option 2: AI-Enhanced Version (gemini.html)
1. Get a free Gemini API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Open `gemini.html` and replace `const apiKey = "";` with your API key
3. Open in browser and enjoy AI-powered features!

## 💝 Customization

Want to personalize it? Edit these parts:

1. **Anniversary Date**: Change `October 15, 2023` in the JavaScript
2. **Photos**: Replace images in the `assets/` folder
3. **Messages**: Edit the text in HTML or the `noTexts` array
4. **Colors**: Modify Tailwind classes (red-400, pink-300, etc.)
